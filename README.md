# How to use:
1. Open the index.html file and copy the code, or download the file. (You can also fork this repo and use it from there)                      
2. If you then have the HTML code, you can paste it into an HTML compiler, such as onecompiler, or if you forked this repo, you can also deploy it on GitHub or Vercel.
